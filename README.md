# Our Love Story – Hosny & Nada

A romantic single-page Valentine’s site.

## Images

Place your memory photos in the **`assets`** folder with these names:

- `image1.jpeg`
- `image2.jpeg`
- `image3.jpeg`
- `image4.jpeg`
- `image5.jpeg`

They will appear in the “Our Best Memories” slider in order.

## Deploy on Vercel

1. Push this folder to a Git repository (e.g. GitHub).
2. Go to [vercel.com](https://vercel.com) and sign in.
3. Click **Add New** → **Project** and import your repo.
4. Leave **Build Command** and **Output Directory** empty (Vercel will treat it as a static site).
5. Click **Deploy**.

Your site will be live at `https://your-project.vercel.app`.

### Deploy from your computer (Vercel CLI)

```bash
npm i -g vercel
vercel
```

Follow the prompts and deploy from this folder.
